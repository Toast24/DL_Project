# Prior Work Basis

This file lists the core papers studied and briefly states how each paper influenced the project.

- Bootstrap Fine-Grained Vision-Language Alignment for Unified Zero-Shot Anomaly Localization (Ano_CLIP): informed our fine-grained vision-language alignment and zero-shot anomaly localization strategy, especially for prompt-driven detection in CLIP-style models.
- Towards Zero-Shot Anomaly Detection and Reasoning with Multimodal Large Language Models (Anomaly_OV_MLLM): influenced our use of multimodal LLM reasoning and prompt design for anomaly detection, linking visual inputs to higher-level anomaly explanations.
- SSVP: Synergistic Semantic-Visual Prompting for Industrial Zero-Shot Anomaly Detection (Anomaly_SSVP_MLLM): guided our semantic-visual prompt design and supported experiments combining vision and language prompts for robust industrial anomaly detection.
- AnomalyVFM – Transforming Vision Foundation Models into Zero-Shot Anomaly Detectors (Anomaly_VFM_MLLM): influenced our approach to converting foundation vision models into anomaly detectors and reinforced the zero-shot evaluation methodology.
- Zero-Shot Anomaly Detection with Dual-Branch Prompt Selection (Prompt Selection Methodology): informed our prompt selection methodology and evaluation of prompt variants for zero-shot anomaly detection.
- AA-CLIP: Enhancing Zero-Shot Anomaly Detection via Anomaly-Aware CLIP (YESAA_CLIP): contributed ideas for anomaly-aware prompt scoring and CLIP adaptation to improve zero-shot anomaly detection robustness.
- AFR-CLIP: Enhancing Zero-Shot Industrial Anomaly Detection with Stateless-to-Stateful Anomaly Feature Rectification (YESAFR_CLIP): motivated feature rectification and robustness techniques in anomaly feature extraction.
- Anomaly Detection by Adapting a Pre-trained Vision Language Model (YESAda_CLIP): influenced our adaptation of pre-trained VLMs for anomaly detection and the use of prompt-based model adaptation strategies.
- AnomalyCLIP: Object-agnostic Prompt Learning for Zero-shot Anomaly Detection (YESAnomaly_CLIP): supported our object-agnostic prompt learning approach and zero-shot anomaly detection framework.
- Prompt as Knowledge Bank: Boost Vision Language Model via Structural Representation for Zero-Shot Medical Detection (YESBoost VLM using Prompt bank): inspired prompt-bank-style methods and structured prompt selection for improving VLM detection performance.
- Bulk to Budget LLM (YESBulk to Budget LLM): informed our efficiency-aware LLM usage and compression mindset, especially when balancing performance with inference cost.
- FineLIP: Extending CLIP’s Reach via Fine-Grained Alignment with Longer Text Inputs (YESExtending_CLIPs_Fine-Grained_Al): influenced our fine-grained prompt alignment and use of longer, richer text inputs for VLM guiding.
- Domain Adaptation of Attention Heads for Zero-shot Anomaly Detection (YESHead_CLIP): guided our attention-head adaptation and compression experiments, particularly in the context of zero-shot anomaly tasks.
- Task Specific Pruning with LLM-Sieve: How Many Parameters Does Your Task Really Need? (YESLLM_Pruning_Methodology): informed our LLM pruning and compression pipeline, including the evaluation of task-specific parameter reductions.
- WinCLIP: Zero-/Few-Shot Anomaly Classification and Segmentation (YESWin_CLIP): provided a zero/few-shot anomaly detection and segmentation baseline that shaped our evaluation and comparison framework.
- A Survey on Zero-Shot Image Anomaly Detection (ZSIAD_Survey): provided the broader domain survey and prior-work basis for framing our zero-shot anomaly detection contributions.
